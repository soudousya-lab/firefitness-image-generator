# 🔥 FIREFITNESS 画像生成ツール

パーソナルトレーニングジム「FIREFITNESS」のマーケティング用画像を、簡単な日本語入力から自動生成するツールです。

## 特徴

- **簡単操作**: 選択式UI + 自由入力で直感的に操作
- **ブランド準拠**: FIREFITNESSのガイドラインを自動反映
- **参照画像対応**: 店舗背景・トレーナーの外見を維持して生成
- **Claude + Gemini連携**: 日本語を最適な英語プロンプトに自動変換

## システム構成

```
[日本語入力] → [Claude API: プロンプト最適化] → [Gemini API: 画像生成]
```

## セットアップ

### 1. 必要なパッケージをインストール

```bash
pip install -r requirements.txt
```

### 2. 初期セットアップを実行

```bash
python setup.py
```

### 3. APIキーを設定

`.env.example` を `.env` にコピーして、APIキーを設定：

```bash
cp .env.example .env
```

`.env` を編集：
```
GEMINI_API_KEY=your_gemini_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key
```

### 4. 画像ファイルを配置

```
assets/
├── trainers/
│   ├── okada/        ← 岡田さんの写真
│   ├── yamamoto/     ← 山本さんの写真
│   ├── itakura/      ← 板倉さんの写真
│   └── kuzumoto/     ← 葛本さんの写真
└── backgrounds/
    ├── shimadahonmachi/  ← 島田本町店の内観
    └── ifukucho/         ← 伊福町店の内観
```

対応画像形式: `.jpg`, `.jpeg`, `.png`, `.webp`

## 使い方

### アプリを起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

### 操作手順

1. **サイドバー**で店舗とトレーナーを選択
2. **メインエリア**でシチュエーション、クライアント、サイズを選択
3. **追加指示**に自由にテキストを入力（オプション）
4. **「画像を生成する」**ボタンをクリック
5. 生成された画像をダウンロード

## 選択オプション

### シチュエーション
| 名前 | 説明 |
|------|------|
| カウンセリング・相談 | 初回相談、プラン説明 |
| 姿勢チェック・診断 | 3軸診断の姿勢評価 |
| セッション風景 | 落ち着いたトレーニング |
| 食事相談・説明 | 栄養指導 |
| 施設内観 | 人物なし |
| 図解・インフォグラフィック | 説明用 |
| 目標達成で喜ぶ風景 | 成功事例 |

### サイズ
| アスペクト比 | 用途 |
|--------------|------|
| 1:1 | Instagram投稿 |
| 4:5 | Instagram縦長 |
| 16:9 | YouTube, 横長広告 |
| 9:16 | ストーリー, リール |
| 4:3 | チラシ |

### 解像度
| 解像度 | 推奨用途 |
|--------|----------|
| 1K | SNS用、プレビュー |
| 2K | 一般的なマーケティング |
| 4K | 印刷物、大型広告 |

## ブランドガイドライン（自動適用）

このツールは以下のガイドラインを自動的に反映します：

### ✅ OK（目指すビジュアル）
- 落ち着いた空間での会話シーン
- 姿勢チェック、カウンセリング風景
- 自然光、清潔感
- 「考えている」「説明を聞いている」表情

### ❌ NG（避けるビジュアル）
- ムキムキ筋肉、汗だく、叫ぶトレーニング
- ギラギラした色（金赤黒）
- 「限界突破」系の煽り
- 過度な体型ビフォーアフター

### カラー
- メイン: #0d2b45（ダークネイビー）
- アクセント: #ff6b35（オレンジ）

## トラブルシューティング

### 画像が生成されない
- APIキーが正しく設定されているか確認
- Gemini APIの利用制限を確認

### 参照画像が表示されない
- 画像ファイルが正しいフォルダにあるか確認
- ファイル形式が対応しているか確認

### エラーが表示される
- `python setup.py` で環境を再確認
- 必要なパッケージがインストールされているか確認
